Blog-Hexo-一

Author: 33 Date: March 31, 2020  16:33:17 Category: Hexo

此文介绍Hexo建站的流程操作

基于B站视频

windows 10系统 以下所有内容均在win10 64位下演示操作

特别感谢 up主 CodeSheep

开整!

01.安装 Node.js

打开官网

https://nodejs.org

然后我们选择Windows版本下载

安装步骤简单,官方默认就好,一直Next.

02.安装 Git

打开官网

https://git-scm.com/downloads

然后我们选择Windows版本下载

安装也是一直下一步,完成后可以在开始菜单看到.

Tips: 这个Git Bash相当于Linux中的终端窗口,以后我们用这个东西打开终端.

打开终端输入node -v npm -v 查看是否安装成功

03.安装Hexo

安装Hexo之前我们需要安装cnpm提高速度(国内镜像源速度很慢),利用npm来安装cnpm

在终端输入npm install -g cnpm -- registry=https://registry.npm.taobao.org

完成安装之后安装Hexo

cnpm install -g hexo-cli

输入hexo -v验证是否安装成功

出现,说明成功

Hexo安装完成后,输入mkdir sansan创建名为sansan的文件夹,位置在E:

随后输入hexo init初始化一个博客

这里我们有几个常用指令
