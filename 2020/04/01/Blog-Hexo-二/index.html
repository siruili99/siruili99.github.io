Blog-Hexo-二

Author: 33 Date: April 1, 2020  11:41:17 Category: Blog

此文介绍Hexo搭建好后如何将博客部署到GitHub上公开使用

01.登陆GitHub

打开官网

https://github.com/

登陆完成

02.新建一个仓库

用户部署个人博客的github仓库的命名必须符合特定要求才可以

创建仓库名称

输入完成后点击Create确认

03.安装插件

打开终端输入npm install hexo-deployer-git --save

下载完成后进入E:/blog/_config.yml设置yml文件

添加type repo branch这三行

repo为你Github的地址，打开刚才创建好的仓库,复制粘贴即可.

04.部署到远端

这一步极其简单

打开终端输入hexo d

推送成功后,刷新仓库

这些其实就是我们博客生成的内容.

打开浏览器输入https://siruili99.github.io进行访问

部署完成后,以后就可以不需要localhost:4000这个地址去访问，通过GitHub地址进行访问.

其余内容请看下篇文章
